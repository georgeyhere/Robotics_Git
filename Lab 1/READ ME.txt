Make sure you have following files in your directory, in order to run the various examples:

1. sim.py
2. simConst.py
3. the appropriate remote API library: "remoteApi.dll" (Windows), "remoteApi.dylib" (Mac) or "remoteApi.so" (Linux)
4. simpleTest.py (or any other example file)

Instructions:

1) Make sure the port numbers in the child scripts in Coppelia match up with the port number at the start of UR3_control.
2) Start Coppelia simulation first.
3) Run ur3_control.py
